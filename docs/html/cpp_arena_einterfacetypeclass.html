EInterfaceTypeClass¶

EInterfaceTypeClass¶

class EInterfaceTypeClass¶

EInterfaceTypeClass is a static class based on GenApi classes of the same purpose. It translates from an interface type enum (GenApi::EInterfaceType) to a string and back. For example, the following enum value can be translated to and from the following string representation:

  • enum: GenApi::EInterfaceType::intfIValue
  • string: “intfIValue”

Warning
  • Similar classes are found mostly under the GenApi namespace

FromString¶

static bool Arena::EInterfaceTypeClass::FromString(const GenICam::gcstring &ValueStr, GenApi::EInterfaceType *pValue)¶

FromString translates a string representation of an interface type enum to its enum value. The second parameter is an out parameter.

Return
  • Type: bool
  • True if successful
  • Otherwise, false
Parameters
  • ValueStr:
    • Type: const GenICam::gcstring&
    • String representation to translate
  • pValue:
    • Type: GenApi::EInterfaceType*
    • Out parameter
    • Pointer to translated enum value

ToString¶

static void Arena::EInterfaceTypeClass::ToString(GenICam::gcstring &ValueStr, GenApi::EInterfaceType Value)

ToString translates from an interface type enum value to its string representation. The first parameter is an out parameter.

Return
  • none
Parameters
  • ValueStr:
    • Type: GenICam::gcstring&
    • Pass-by-reference out parameter
    • Translated string representation
    • ”intfIUnknown” on failure
  • Value:
    • Type: GenApi::EInterfaceType
    • Enum value to translate
