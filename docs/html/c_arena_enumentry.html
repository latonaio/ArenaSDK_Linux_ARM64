EnumEntry¶

acEnumEntryGetIntValue¶

AC_ERROR AC_API acEnumEntryGetIntValue(acNode hNode, int64_t * pValue)

acEnumEntryGetIntValue retrieves the integer value of the enum entry.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hNode:
    • Type: acNode
    • [In] parameter
    • A node
    • Accepts enum entry nodes
  • pValue:
    • Type: int64_t*
    • [Out] parameter
    • Integer value of the entry
