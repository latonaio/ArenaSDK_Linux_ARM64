acBufferGetSizeFilled¶

AC_ERROR AC_API acBufferGetSizeFilled(acBuffer hBuffer, size_t * pSizeFilled)

acBufferGetSizeFilled retrieves the size of the payload data, excluding transport layer protocol leaders. The payload data may include image data, chunk data, or both.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hBuffer:
    • Type: acBuffer
    • [In] parameter
    • A buffer
  • pSizeFilled:
    • Type: size_t*
    • [Out] parameter
    • Size of the payload

acBufferGetSizeFilled retrieves the size of the data of a buffer, excluding transport layer protocol leaders.

    • Out parameter
    • Size of the payload data

acBufferGetPayloadSize¶

AC_ERROR AC_API acBufferGetPayloadSize(acBuffer hBuffer, size_t * pPayloadSize)

GetPayloadSize retrieves the intended size of the payload. This is similar to the retrieved payload size (acBufferGetSizeFilled), but different in that missed data is included.

Return
  • Type: size_t
  • Unit: bytes
  • Size of the intended payload

Warning
  • Causes undefined behavior if buffer

acBufferVerifyCRC¶

AC_ERROR AC_API acBufferVerifyCRC(acBuffer hBuffer, bool8_t * pVerifyCRC)

acBufferVerifyCRC calculates the CRC of a buffer’s data and verifies it against the CRC value sent from the device via chunk data. This helps verify that no data has been changed or missed during a transmission. This function calls a global helper function to calculate the CRC (acCalculateCRC32).

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hBuffer:
    • Type: acBuffer
    • [In] parameter
    • A buffer
  • pVerifyCRC:
    • Type: bool8_t*
    • [Out] parameter
    • True if the calculated CRC value equals the one sent from the device
    • Otherwise, false

See
  • acCalculateCRC
