VideoRecorder¶

saveRecorderCreateEmpty¶

SC_ERROR SC_API saveRecorderCreateEmpty(saveRecorder * hRecorder)

saveRecorderCreateEmpty creates an empty/default video recorder. The parameters of the video recorder are not yet set (width = height = 0, FPS = 0/0). The default file name pattern is ‘savedvideos/video<count>.avi’. The default codec is raw, the default container is AVI, and the default pixel format is BGR8.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hRecorder:
    • Type: saveRecorder*
    • [Out] parameter
    • Created video recorder

saveRecorderSetFileNamePattern¶

SC_ERROR SC_API saveRecorderSetFileNamePattern(saveRecorder hRecorder, const char * pFileNamePattern)

saveRecorderSetFileNamePattern sets a file name pattern. Changing the extension via this call will change the output container.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hRecorder:
    • Type: saveRecorder
    • [In] parameter
    • A video recorder
  • pFileNamePattern:
    • Type: const char*
    • [In] parameter
    • File name pattern to set

The video recorder defaults to different settings depending on the extension of the file name pattern:

  • ’.avi’ defaults to raw codec, BGR8 pixel format
  • ’.mov’ defaults to raw codec, RGB8 pixel format
  • ’.mp4’ defaults to H264 codec, BGR8 pixel

Warning
  • changes to the file extension will change the container accordingly
  • set file name pattern before setting other settings
  • cannot be called while recording
