Command¶

acCommandExecute¶

AC_ERROR AC_API acCommandExecute(acNode hNode)

acCommandExecute executes an action associated with the command node.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hNode:
    • Type: acNode
    • [In] parameter
    • A node
    • Accepts command nodes

acCommandIsDone¶

AC_ERROR AC_API acCommandIsDone(acNode hNode, bool8_t * pIsDone)

acCommandIsDone retrieves whether or not the execution of the node has completed.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hNode:
    • Type: acNode
    • [In] parameter
    • A node
    • Accepts command nodes
  • pIsDone:
    • Type: bool8_t*
    • [Out] parameter
    • False if the action is currently processing
    • Otherwise, true
