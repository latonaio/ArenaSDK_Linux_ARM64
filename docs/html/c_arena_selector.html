Selector¶

acSelectorIsSelector¶

AC_ERROR AC_API acSelectorIsSelector(acNode hNode, bool8_t * pIsSelector)

acSelectorIsSelector retrieves whether or not a node is a selector.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hNode:
    • Type: acNode
    • [In] parameter
    • A node
    • Accepts selector nodes
  • pIsSelector:
    • Type: bool8_t*
    • [Out] parameter
    • True if node is a selector
    • Otherwise, false
