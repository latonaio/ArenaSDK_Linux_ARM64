GenTL Interface Features¶

Node Type (Access) Description
InterfaceID String (RO) GenTL Producer wide unique identifier of the selected interface.
InterfaceDisplay String (RO) User readable name of the selected interface.
InterfaceType Enumeration (RO) Custom transport layer. The entries for this enumeration node are ‘GigEVision’, ‘CameraLink’, ‘CameraLinkHS’, ‘CoaXPress’, ‘USB3Vision’, and ‘Custom’.
GevInterfaceGatewaySelector Integer (RW) Selector for the different gateway entries for this interface. The selector is 0-based.
GevInterfaceGateway Integer (RO) IP address of the selected gateway entry of this interface.
GevInterfaceMACAddress Integer (RO) 48-bit MAC address of this interface.
GevInterfaceSubnetSelector Integer (RW) Selector for the subnet of this interface.
GevInterfaceSubnetIPAddress Integer (RO) IP address of the selected subnet of this interface.
GevInterfaceSubnetMask Integer (RO) Subnet mask of the selected subnet of this interface.
InterfaceDeviceSelector Integer (RW) Selector for the different devices on this interface. This value only changes on execution of “DeviceUpdateList”. The selector is 0-based in order to match the index of the C interface.
InterfaceDeviceID String (RO) Interface wide unique identifier of the selected device. This value only changes on execution of “DeviceUpdateList”.
InterfaceDeviceVendor String (RO) Name of the device vendor. This value only changes on execution of “DeviceUpdateList”.
InterfaceDeviceModel String (RO) Name of the device model. This value only changes on execution of “DeviceUpdateList”.
InterfaceDeviceAccessStatus Enumeration (RO) Another device has exclusive access. The entries for this enumeration node are ‘ReadWrite’, ‘ReadOnly’, and ‘NoAccess’.
InterfaceDeviceSerialNumber String (RO) Serial number of the remote device. This value only changes on execution of “DeviceUpdateList”.
InterfaceDeviceUserID String (RO) User-programmable device identifier of the remote device. This value only changes on execution of “DeviceUpdateList”.
InterfaceGevDeviceIPAddress Integer (RO) Current IP address of the GVCP interface of the selected remote device.
InterfaceGevDeviceSubnetMask Integer (RO) Current subnet mask of the GVCP interface of the selected remote device.
InterfaceGevDeviceMACAddress Integer (RO) 48-bit MAC address of the GVCP interface of the selected remote device.
