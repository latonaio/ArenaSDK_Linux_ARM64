acOpenSystem¶

AC_ERROR AC_API acOpenSystem(acSystem * phSystem)

acOpenSystem initializes the Arena SDK and retrieves the system object (acSystem). The system must be closed or memory will leak.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • phSystem:
    • Type: acSystem*
    • [Out] parameter
    • The system object

  • acSystem

acGetBitsPerPixel¶

AC_ERROR AC_API acGetBitsPerPixel(uint64_t pixelFormat, size_t * pBitsPerPixel)

acGetBitsPerPixel get the number of bits per pixel of the image from the integer value of the pixel format (PfncFormat).

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • pixelFormat:
    • Type: uint64_t
    • [In] parameter
    • Pixel format to get bits per pixel
  • pBitsPerPixel:
    • Type: size_t*
    • [Out] parameter
    • Number of bits per pixel for given pixel format

acCalculateCRC32¶

AC_ERROR AC_API acCalculateCRC32(const uint8_t * pData, size_t pDataLen, size_t * pCRCValue)

acCalculatedCRC32 calculates a CRC value (cyclical redundancy check) on a dataset. This is used to check whether the dataset has been sent in its entirety.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • pData:
    • Type: const uint8_t*
    • [In] parameter
    • A pointer to the data to use to calculate the CRC
  • pDataLen:
    • Type: size_t
    • [In] parameter
    • The size of the data
  • pCRCValue:
    • Type: size_t*
    • [Out] parameter
    • The calculated CRC value

acCalculateMaximumNumberOfBuffers¶

AC_ERROR AC_API acCalculateMaximumNumberOfBuffers(size_t payloadSize, size_t * pMaxBufs)

acCalculateMaximumNumberOfBuffers calculates the number of buffers it would take to fill 80% of the available memory.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • payloadSize:
    • Type: size_t
    • Unit: bytes
    • [In] parameter
    • Payload size of an image
  • pMaxBufs:
    • Type: size_t*
    • [Out] parameter
    • Maximum number of buffers
