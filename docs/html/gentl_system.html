GenTL System Features¶

Node Type (Access) Description
TLID String (RO) Unique identifier of the GenTL Producer like a GUID.
TLVendor String (RO) Name of the GenTL Producer vendor.
TLModel String (RO) Name of the GenTL Producer to distinguish different kinds of GenTL Producer implementations from one vendor.
TLVersion String (RO) Vendor specific version string.
TLFile String (RO) Filename including extension of the GenTL Producer.
TLDisplay String (RO) User readable name of the GenTL Producer.
TLPath String (RO) Full path to the GenTL Producer including filename and extension.
TLType Enumeration (RO) Custom transport layer. The entries for this enumeration node are ‘GigEVision’, ‘CameraLink’, ‘CameraLinkHS’, ‘CoaXPress’, ‘USB3Vision’, ‘Mixed’, and ‘Custom’.
GenTLVersionMajor Integer (RO) Major version number of the GenTL specification the GenTL Producer implementation complies with.
GenTLVersionMinor Integer (RO) Minor version number of the GenTL specification the GenTL Producer implementation complies with.
GenTLSFNCVersionMajor Integer (RO) Major version number of the GenTL Standard Features Naming Convention that was used to create the GenTL Producer`s XML.
GenTLSFNCVersionMinor Integer (RO) Minor version number of the GenTL Standard Features Naming Convention that was used to create the GenTL Producer`s XML.
GenTLSFNCVersionSubMinor Integer (RO) Sub minor version number of the GenTL Standard Features Naming Convention that was used to create the GenTL Producer`s XML.
GevVersionMajor Integer (RO) Major version number of the GigE Vision specification the GenTL Producer implementation complies to.
GevVersionMinor Integer (RO) Minor version number of the GigE Vision specification the GenTL Producer implementation complies to.
InterfaceSelector Integer (RW) Selector for the different GenTL Producer interfaces. This interface list only changes on execution of “InterfaceUpdateList”. The selector is 0-based in order to match the index of the C interface.
InterfaceID String (RO) GenTL Producer wide unique identifier of the selected interface. This interface list only changes on execution of “InterfaceUpdateList”.
GevInterfaceMACAddress Integer (RO) 48-bit MAC address of the selected interface.
GevInterfaceDefaultIPAddress Integer (RO) IP address of the first subnet of the selected interface.
GevInterfaceDefaultSubnetMask Integer (RO) Subnet mask of the first subnet of the selected interface.
ActionCommandDeviceKey Integer (RW) User-specified Action Command Device Key.
ActionCommandGroupKey Integer (RW) User-specified Action Command Group Key.
ActionCommandGroupMask Integer (RW) User-specified Action Command Group Mask.
ActionCommandTargetIP Integer (RW) Target IP Address for the action command.
ActionCommandExecuteTime Integer (RW) Time when the device will execute the command.
ThreadSelection Enumeration (RW) Heartbeat Thread. The entries for this enumeration node are ‘GrabThread’, ‘EventThread’, and ‘HeartbeatThread’.
ThreadPriority Enumeration (RW) Critical Priortiy. The entries for this enumeration node are ‘BelowNormal’, ‘Normal’, ‘AboveNormal’, and ‘Critical’.
