acSystem¶

typedef acSystem¶

Representation of the system object, the entry point into Arena SDK.

acDevice¶

typedef acDevice¶

Represents a device, used to configure and stream a device.

acBuffer¶

typedef acBuffer¶

Represents a buffer, which carries a payload of either an image or a chunk data. A buffer can hold image data and/or chunk data.

acFeatureStream¶

typedef acFeatureStream¶

Represents a feature stream object, allowing node map configurations to be streamed to and from files.

acNodeMap¶

typedef acNodeMap¶

Represents a node map, or feature tree.

acNode¶

typedef acNode¶

Represents a node, or feature.

acCallback¶

typedef acCallback¶

Represents a callback handle, used to deregister callbacks when they are no longer needed.

acCallbackFunction¶

typedef acCallbackFunction¶

The signature of a function needed to register a callback. An example of a function that could work might look something like the following.

// a function with the appropriate function signature to register as a callback
// error handling skipped for brevity
{
    void onCallback(acNode hNode, void* pUserData)
    {
        UserDataStruct* userData = reinterpret_cast<UserDataStruct*>(pUserData);
        
        char pBuf[100];
        size_t len = 100;
        acNodeGetName(hNode, pBuf, &len);
        // ...
    }
}
