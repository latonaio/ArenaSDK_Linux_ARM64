ImageFactory¶

acImageFactoryCreate¶

AC_ERROR AC_API acImageFactoryCreate(uint8_t * pData, size_t dataSize, size_t width, size_t height, uint64_t pixelFormat, acBuffer * phDst)

FactoryCreate(uint8_t pData, size_t dataSize, size_t width, size_t height, uint64_t pixelFormat, acBuffer* phDst)

acImageFactoryCreate creates an image (acBuffer) from a minimal set of parameters. Images created with the image factory must be destroyed (acImageFactoryDestroy) when no longer needed.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • pData:
    • Type: uint8_t*
    • [Out] parameter
    • Pointer to the beginning of the payload data
  • dataSize:
    • Type: size_t
    • [In] parameter
    • Size of the data
  • width:
    • Type: size_t
    • [In] parameter
    • Width of the image to create
  • height:
    • Type: size_t
    • [In] parameter
    • Height of the image to create
  • pixelFormat:
    • Type: uint64_t
    • [In] parameter
    • Pixel format of the image to create
  • phDst:
    • Type: acBuffer*
    • [Out] parameter
    • Image created from the parameters

acImageFactoryCopy¶

AC_ERROR AC_API acImageFactoryCopy(acBuffer hSrc, acBuffer * phDst)

acImageFactoryCopy creates a deep copy of an image (acBuffer) from another image. Images created with the image factory must be destroyed (acImageFactoryDestroy) when no longer needed.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hSrc:
    • Type: acBuffer
    • [In] parameter
    • Image to copy
  • phDst:
    • Type: acBuffer*
    • [Out] parameter
    • Deep copy of the image

acImageFactoryConvert¶

AC_ERROR AC_API acImageFactoryConvert(acBuffer hSrc, uint64_t pixelFormat, acBuffer * phDst)

acImageFactoryConvert converts an image (acBuffer) to a select pixel format. In doing so, it creates a completely new image, similar to a deep copy but with a different pixel format. Images created with the image factory must be destroyed (acImageFactoryDestroy) when no longer needed; otherwise, memory will leak.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hSrc:
    • Type: acBuffer
    • [In] parameter
    • Image to convert
  • pixelFormat:
    • Type: uint64_t
    • [In] parameter
    • Pixel format to convert to
  • phDst:
    • Type: acBuffer*
    • [Out] parameter
    • Convert image

See
  • acBuffer
  • acImageFactoryDestroy

acImageFactoryDestroy¶

AC_ERROR AC_API acImageFactoryDestroy(acBuffer hBuffer)

acImageFactoryDestroy cleans up an image (acBuffer) and deallocates its memory. It must be called on any image created by the image factory (acImageFactoryCreate, acImageFactoryCopy, acImageFactoryConvert).

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hBuffer:
    • Type: acBuffer
    • [In] parameter
    • Image to destroy
    • Image must be from image factory
