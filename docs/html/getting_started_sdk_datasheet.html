Arena SDK Datasheet¶

LUCID Arena SDK for Windows¶

Data Interface Gigabit Ethernet (1000 Mbit/s)
Operating System Windows 7 / 10 (32-bit and 64-bit)
Compiler Visual Studio 2015 project files included.
Programming Languages C++, C, and C#
Supported Standards GenICam 3, GenTL 1.5, SFNC 2.3, GigE Vision 2.0
Supported Cameras All LUCID GigE Vision cameras

LUCID Arena SDK for Linux¶

Data Interface Gigabit Ethernet (1000 Mbit/s)
Operating System Ubuntu 16.04 / 18.04 (64-bit)
Compiler G++ 5 or higher
Kernel Ubuntu 16.04: 4.10.0-28 or higher, Ubuntu 18.04: 4.15.0-20 or higher
Programming Languages C++ and C
Supported Standards GenICam 3, GenTL 1.5, SFNC 2.3, GigE Vision 2.0
Supported Cameras All LUCID GigE Vision cameras

Minimum Requirements (x86 or x64 platforms)¶

CPU Intel Core i3-4340 CPU @ 3.60 GHz (Haswell)
Memory 4 GB
Network Ethernet adapter required (dedicated Ethernet adapter recommended)

Recommended Requirements¶

Single camera system¶

CPU Intel Core i5 from Skylake series or higher
Memory 8 GB or higher
Network Dedicated Intel Ethernet adapter

Multi-camera system¶

CPU Intel Core i7 from Skylake series or higher
Memory 8 GB or higher
Network Dedicated Intel Ethernet adapter(s)

Documentation API Reference for C++, C, and C#
