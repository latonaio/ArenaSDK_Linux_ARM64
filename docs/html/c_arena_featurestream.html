FeatureStream¶

acFeatureStreamCreate¶

AC_ERROR AC_API acFeatureStreamCreate(acNodeMap hNodeMap, acFeatureStream * phFeatureStream)

acFeatureStream builds a feature stream from a device’s node map (acNodeMap). This node map cannot be changed later; another feature stream must be created to stream another node map.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hNodeMap:
    • Type: acNodeMap
    • [In] parameter
    • Node map to stream
  • phFeatureStream:
    • Type: acFeatureStream*
    • [Out] parameter
    • Feature stream object to create

See
  • acNodeMap

acFeatureStreamSelect¶

AC_ERROR AC_API acFeatureStreamSelect(acFeatureStream hFeatureStream, const char * pFeatureName)

acFeatureStreamSelect adds a single streamable feature to the list of selected features to stream. If called for the first time, acFeatureStreamSelect also sets an internal ‘select-all’ flag to false.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hFeatureStream:
    • Type: acFeatureStream
    • [In] parameter
    • The feature stream object
  • pFeatureName:
    • Type: const char*
    • [In] parameter
    • Name of the feature to select
