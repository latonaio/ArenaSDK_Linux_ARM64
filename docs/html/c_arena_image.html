Image¶

acImageGetWidth¶

AC_ERROR AC_API acImageGetWidth(acBuffer hBuffer, size_t * pWidth)

acImageGetWidth gets the width of the image in pixels. Images are self-describing, so the device does not need to be queried to get this information.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hBuffer:
    • Type: acBuffer
    • In parameter
    • An image
  • pWidth:
    • Type: size_t*
    • [Out] parameter
    • Width of the image
