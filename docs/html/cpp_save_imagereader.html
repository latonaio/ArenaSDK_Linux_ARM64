class ImageReader¶

The image reader facilitates the loading of image data from files.

ImageReader¶

Save::ImageReader::ImageReader(const char *pFileName, bool topToBottom = true)

A constructor. Internally, it loads an image (

Save::ImageReader::Load).
Parameters
  • pFileName:
    • Type: const char*
    • Name of the file to load
  • topToBottom:
    • Type: Type: bool
    • Default: true
    • If true, read the file from top to bottom
    • Otherwise, from bottom to top

Load¶

void Save::ImageReader::Load(const char *pFileName, bool topToBottom = true)¶

Load loads image data from a file to memory. After loading an image, the image parameters can be retrieved (Save::ImageReader::GetParams) or the image data (Save::ImageReader::GetData).

Return
  • none
Parameters
  • pFileName:
    • Type: const char*
    • Name of the file to load

LoadRawData¶

static void Save::ImageReader::LoadRawData(const char *pFileName, uint8_t *pImageData, const size_t size)¶

LoadRawData is a static function that loads the raw data of a file.

Parameters
  • pFileName:
    • Type: const char*
    • Name of the file to load
  • pImageData:
    • Type: uint8_t*
    • Out parameter
    • Requires a preallocated buffer
    • Returns data from the loaded file
  • size:
    • Type: const size_t
    • Size of the file to read

Warning
  • Reads files top to bottom
