acCategoryGetNumFeatures¶

AC_ERROR AC_API acCategoryGetNumFeatures(acNode hNode, size_t * pNumFeatures)

acCategoryGetNumFeatures retrieves the number of features of a category node.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success
Parameters
  • hNode:
    • Type: acNode
    • [In] parameter
    • A node
    • Accepts category nodes
  • pNumFeatures:
    • Type: size_t*
    • [Out] parameter
    • Number of features
