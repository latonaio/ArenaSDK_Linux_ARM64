acSystemUpdateDevices¶

AC_ERROR AC_API acSystemUpdateDevices(acSystem hSystem, uint64_t timeout)

acSystemUpdateDevices updates the internal list of devices, (along with their relevant interfaces). It must be called before retrieving the number of devices (acSystemGetNumDevices) or any time that an updated device list might be necessary.

Return
  • Type: AC_ERROR
  • Error code for the function
  • Returns AC_ERR_SUCCESS (0) on success

    • Type: acSystem
    • [In] parameter
    • The system object
  • timeout:
    • Type: uint64_t
    • [In] parameter
    • Time to wait for connected devices to respond
