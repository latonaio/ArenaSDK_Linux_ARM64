ImageWriter¶

saveWriterCreateEmpty¶

SC_ERROR SC_API saveWriterCreateEmpty(saveWriter * phWriter)

saveWriterCreateEmpty creates an empty/default image writer. The parameters of this image writer are not set (height = width = bits per pixel = 0). The default file name pattern is ‘savedimages/image<count>.jpg’ and the default file format is jpeg.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • phWriter:
    • Type: saveWriter*
    • [Out] parameter
    • Created image writer

Warning
  • unable to save before setting parameters

saveWriterCreate creates an image writer while setting the parameters of the upcoming image(s) to save. The default file name pattern is ‘savedimages/image<count>.jpg’ and the default file format is jpeg.

saveWriterSetJpegAndConfig¶

SC_ERROR SC_API saveWriterSetJpegAndConfig(saveWriter hWriter, size_t quality, bool8_t progressive, SC_JPEG_SUBSAMPLING subsampling, bool8_t optimize)

saveWriterSetJpeg changes the image writer to save images using the jpeg file format. It also switches the extension to ‘.jpg’.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hWriter:
    • Type: saveWriter
    • [In] parameter
    • An image writer
  • quality:
    • Type: size_t
    • [In] parameter
    • Range: 1-100
    • Image quality (1 lower, 100 highest)
  • progressive:
    • Type: bool8_t
    • [In] parameter
    • If true, saves progressive
    • Otherwise, saves baseline
  • subsampling:
    • Type: SC_JPEG_SUBSAMPLING
    • [In] parameter
    • The chroma subsampling to apply
  • optimize:
    • Type: bool8_t
    • [In] parameter
    • If true, calculates optimal Huffman coding tables
    • Otherwise, does not

SC_ERROR SC_API saveWriterSetPlyAndConfig(saveWriter hWriter, bool8_t filterPoints)

  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hWriter:
    • Type: saveWriter
    • [In] parameter

  • filterPoints:
    • Type: bool8_t,
    • [In] parameter
    • Default: true
    • Filter NaN points (A = B = C = -32,678)

saveWriterSetPlyAndConfigExtended¶

SC_ERROR SC_API saveWriterSetPlyAndConfigExtended(saveWriter hWriter, savePlyParams params)

saveWriterSetPlyAndConfigExtended changes the image writer to save images as ply data. It switches the extension to ‘.ply’ and uses savePlyParams information to configre the save data. The param.filterPoints option will filter NaN points in the output data. The params.scale, and params.offset(A/B/C) will be applied to the output PLY file accordingly also. Finally the params.isSigned value lets the library know if data passed in is signed or unsigned.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hWriter:
    • Type: saveWriter
    • [In] parameter
    • An image writer
  • params:
    • Type: savePlyParams,
    • [In] parameter
    • Passes savePlyParams to saveWriterSetPlyAndConfigExtended

saveWriterSetFileNamePattern¶

SC_ERROR SC_API saveWriterSetFileNamePattern(saveWriter hWriter, const char * pFileNamePattern)

saveWriterSetFileNamePattern sets a new file name or pattern to use to save images. Include ‘<count>’ to include an image count (specific to this image image writer). Include ‘<timestamp>’ to include a timestamp. An image’s timestamp must be updated manually.

    • Type: saveWriter
    • [In] parameter

    • Type: const char*
