ImageReader¶

saveReaderCreate¶

SC_ERROR SC_API saveReaderCreate(saveReader * phReader, const char * pFileName)

saveReaderCreate initializes an image reader and loads an image. The image reader must be destroyed or memory will leak. This image reader loads images starting from the upper leftmost pixel.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • phReader:
    • Type: saveReader*
    • [Out] parameter
    • Created image reader
  • pFileName:
    • Type: const char*
    • [In] parameter
    • Relative path to a file to load

saveReaderDestroy¶

SC_ERROR SC_API saveReaderDestroy(saveReader hReader)

saveReaderDestroy cleans up an image reader (saveReader), deallocating its memory.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hReader:
    • Type: saveReader
    • [In] parameter
    • An image reader

Warning
  • Invalidates pointers to image data

saveReaderGetParams¶

SC_ERROR SC_API saveReaderGetParams(saveReader hReader, size_t * pWidth, size_t * pHeight, size_t * pBitsPerPixel)

saveReaderGetParams retrieves the width, height, and number of bits per pixel of the loaded image.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hReader:
    • Type: saveReader
    • [In] parameter
    • An image reader
  • pWidth:
    • Type: size_t*
    • [Out] parameter
    • Width of the loaded image
  • pHeight:
    • Type: size_t*
    • [Out] parameter
    • Height of the loaded image
  • pBitsPerPixel:
    • Type: size_t*
    • [Out] parameter
    • Number of bits per pixel of the loaded image

saveReaderGetData¶

SC_ERROR SC_API saveReaderGetData(saveReader hReader, uint8_t ** ppData)

saveReaderGetData retrieves a pointer to the beginning of the loaded image data. Data must be copied manually in order to retain data beyond beyond the life cycle of the image reader.

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • hReader:
    • Type: saveReader
    • [In] parameter
    • An image reader
  • ppData:
    • Type: uint8_t**
    • [Out] parameter
    • Pointer to the image data

Warning
  • Pointer invalidated on destruction of image reader

saveReaderLoadRawData¶

SC_ERROR SC_API saveReaderLoadRawData(const char * pFileName, uint8_t * pImageData, const size_t size)

saveReaderLoadRawData loads raw data into a buffer. It does not require an image reader context (saveReader).

Return
  • Type: SC_ERROR
  • Error code for the function
  • Returns SC_ERR_SUCCESS (0) on success
Parameters
  • pFileName:
    • Type: const char*
    • [In] parameter
    • Relative path to the file to load
  • pImageData:
    • Type: uint8_t*
    • [Out] parameter
    • Retrieved data
    • Buffer must be preallocated
  • size:
    • Type: const size_t
    • [In] parameter
    • Size of the buffer

Warning
  • does not require context
  • image data buffer must be preallocated

  • saveReader
